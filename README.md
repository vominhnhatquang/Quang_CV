- `outputs/Vo_Minh_Nhat_Quang_CV.docx` — editable Microsoft Word version.
- `outputs/Vo_Minh_Nhat_Quang_CV.md` — source Markdown version.

## Deploy website

1. Push this repository to GitHub.
2. In repository settings, enable GitHub Pages with **GitHub Actions** as the source.
3. Push to `main` or run the `Deploy static CV site` workflow manually.

Private repositories are described in the CV without public links; public GitHub/Hugging Face artifacts are linked.
